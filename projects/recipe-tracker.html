Recipe Tracker

Oct 2023 - Present

I decided to create Recipe Tracker with the goal of helping people find and keep track of their favorite recipes. I know how stressful it is to think of recipes for the week, and I would love to help reduce this stress through Recipe Tracker. This project also has some personal motivation behind it, as I want to be able to save my mom's recipes. If you want to see the source code for this project, you can find it at this link here.
